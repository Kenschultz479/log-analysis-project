# Alert Triage Lab

## Overview
This lab simulates a SOC alert triage workflow by reviewing alerts, assigning severity, and recommending actions.

## Objective
Demonstrate understanding of security alert prioritization and incident response workflow.

## Files
- `alerts.csv` - sample security alerts with severity levels
- `alert_triage.py` - Python script that reads alerts and recommends next steps
- `triage_playbook.txt` - simple triage guidance by severity
- `ticket_example.md` - example SOC ticket documentation

## Skills Demonstrated
- Alert triage
- Incident prioritization
- Security workflow awareness
- Documentation
- Python scripting

## Example Use Cases
- Failed login bursts
- Admin login review
- Malware signature alert
- Port scan detection

## Outcome
The lab shows how different alert severities can be handled using a repeatable triage process.
