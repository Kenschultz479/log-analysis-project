# Phishing Analysis Lab

## Overview
This lab analyzes a simulated phishing email to identify indicators of malicious intent.

## Objective
Practice identifying common phishing traits such as spoofed domains, urgent language, malicious links, and credential theft attempts.

## Files
- `suspicious_email.txt` - simulated phishing email sample
- `phishing_checker.py` - Python script that checks for common phishing indicators
- `phishing_analysis.md` - written analysis of the phishing email
- `phishing_checklist.txt` - checklist of common phishing indicators

## Skills Demonstrated
- Phishing analysis
- Threat detection
- Security awareness
- Documentation
- Python scripting

## Indicators Identified
- Suspicious sender domain
- Brand impersonation
- Urgent language
- Account lockout threat
- Suspicious link
- Credential request

## Outcome
This lab demonstrates the ability to analyze phishing content and recommend appropriate response actions.
